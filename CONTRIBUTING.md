# Contributing to AI Petition Hub

Thank you for your interest in contributing to the AI Petition Hub! We welcome contributions from the community to help make digital governance more transparent and efficient.

## 🚀 How to Contribute

### 1. Fork and Clone
- Fork the repository on GitHub.
- Clone your fork to your local machine.

### 2. Set Up the Project
- Follow the instructions in the [README.md](README.md) to set up your environment.
- Create a new branch for your feature or bug fix:
  ```bash
  git checkout -b feature/your-feature-name
  ```

### 3. Make Changes
- Implement your changes.
- Ensure your code follows the existing style and is well-documented.
- Write tests if applicable.

### 4. Commit and Push
- Commit your changes with a clear message:
  ```bash
  git commit -m "Add feature: described feature"
  ```
- Push to your fork:
  ```bash
  git push origin feature/your-feature-name
  ```

### 5. Submit a Pull Request
- Open a Pull Request on the main repository.
- Provide a clear description of your changes and why they are needed.

## ⚖️ Code Style
- We use **Prettier** for code formatting.
- Ensure all TypeScript files pass linting.

## 🐞 Reporting Issues
- Use the provided [Issue Templates](.github/ISSUE_TEMPLATE) to report bugs or request features.
- Provide as much detail as possible, including steps to reproduce.
